# Images Directory

Place your images here:

1. **profile.jpg** (or .png) - Your profile picture
   - Recommended size: 400x400px or larger (square)
   - Will be displayed in a circular frame in the Hero section

2. **background.jpg** (or .png) - Background image for Hero section (OPTIONAL)
   - Recommended size: 1920x1080px or larger
   - Can be configured in different styles (see BACKGROUND_OPTIONS.md)

## Image Requirements:
- **Profile Picture**: Square format, high quality, professional headshot
- **Background**: Wide format, subtle/abstract design that doesn't distract from text

## Supported Formats:
- JPG/JPEG
- PNG
- WebP (recommended for better performance)

## Background Image Configuration:

Edit `data/portfolio.ts` to choose your background style:
- `"none"` - No background image (default)
- `"full"` - Full background with opacity
- `"behind-profile"` - Only behind profile picture
- `"gradient-overlay"` - With dark gradient overlay
- `"corner-accent"` - Small accent in corner

See `BACKGROUND_OPTIONS.md` for detailed descriptions.
